# Dashboard Navigation

## Paths and shell behavior

- Dashboard route: `/dashboard/index`
- Sidebar navigation uses module names from the existing shell.
- Sidebar search filters visible items before navigation.
- Quick launch actions should still keep the user in the same app shell.
