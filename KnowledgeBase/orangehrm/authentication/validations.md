# Authentication Validations

## Validation rules to capture

- Login button should be visible.
- Username/password fields should be visible.
- Empty username should show username validation.
- Empty password should show password validation.
- Invalid credentials should show an error alert.
- Credential hints and forgot-password link should be visible on the login page.
